Bootstrap: Control Plane Issues Warrant
One-time setup at session start

Control Plane issues a root warrant to the Orchestrator for infrastructure management. This happens once per session, not per-request.

// Control Plane (secure environment) let root_warrant = control.issue_warrant( "manage_infrastructure", &[("cluster", Pattern::new("staging-*")?), ("budget", Range::max(10000.0))], Duration::from_secs(3600), // 1 hour )?;
Warrant ID: tnu_wrt_a1b2c3d4
2
üé≠
Delegation: Orchestrator Attenuates for Worker
Narrows scope + binds to worker identity

Orchestrator creates a narrower warrant for the Worker. Critically, it binds the warrant to the worker's public key (holder binding). This prevents stolen warrants from being used.

// Orchestrator generates worker's keypair let worker_keypair = Keypair::generate(); let worker_warrant = root_warrant.attenuate() .constraint("cluster", Exact::new("staging-web")) .constraint("budget", Range::max(1000.0)) .authorized_holder(worker_keypair.public_key()) // PoP binding! .agent_id("agt_worker_001") .ttl(Duration::from_secs(600)) // 10 min .build(&orchestrator_keypair)?;
Holder-Bound
Agent: agt_worker_001

Request: Worker Calls Backend via Gateway
Worker signs the action (Proof-of-Possession)

Worker wants to upgrade staging-web. Since the warrant is holder-bound, the worker must sign the request to prove it controls the authorized key.

// Worker creates PoP signature let args = hashmap!{ "cluster" => "staging-web", "version" => "1.28.5", "budget" => 500.0 }; let pop_signature = warrant.create_pop_signature( &my_keypair, "manage_infrastructure", &args );
POST /api/v1/infrastructure/upgrade
X-Tenuo-Chain: [base64(root), base64(worker)]

Security: What Gets Blocked

üö´
Stolen Warrant
Attacker steals worker's warrant from logs/traffic.

BLOCKED

Warrant is holder-bound. Without the worker's private key, attacker can't create valid PoP signature.
üö´
Scope Escalation
Worker tries to upgrade prod-web instead of staging-web.

BLOCKED

Constraint check fails: "staging-web" != "prod-web". Gateway returns 403.
üö´
Budget Override
Worker requests $5,000 budget action (limit: $1,000).

BLOCKED

Range constraint violated: 5000 > max(1000). Gateway returns 403.
üö´
Forged Chain
Attacker creates fake warrant with wider permissions.

BLOCKED

Ed25519 signature invalid. Only Control Plane's key can sign valid root warrants.
