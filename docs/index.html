3 lines to protect your tools

Works with LangChain, LangGraph, and any Python agent

from tenuo import root_task, scoped_task, protect_tools

# Wrap your tools once
tools = [read_file, send_email, query_db]
protect_tools(tools)

# Create scoped authority for each task
async with root_task(tools=["read_file"], path="/data/*"):
    async with scoped_task(path="/data/reports/*"):
        result = await agent.run("Summarize Q3 reports")

Why it matters

The attack succeeds. The action doesn't.
