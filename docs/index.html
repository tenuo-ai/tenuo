Protect your tools in 3 steps

Native protection for LangChain, LangGraph, Google ADK, and OpenAI

from tenuo import configure, mint, guard_tools, Capability, Subpath

# 1. Start safely (dev mode auto-generates keys)
configure(dev_mode=True)
guard_tools([read_file, send_email, query_db])

# 2. Scope authority. The agent can ONLY read /data/.
# send_email and query_db are strictly BLOCKED.
async with mint(Capability("read_file", path=Subpath("/data"))):
    await agent.run("Summarize Q3 reports")

Prompt Injections, Bounded

The model gets tricked. The damage stays contained.

Without tenuo

1 User: "Summarize Q3 report"
2 Agent inherits ambient authority (full credentials)
3 Worker reads /data/q3.pdf
4 PDF contains malicious prompt injection
5 Worker has send_email capability (inherited)
