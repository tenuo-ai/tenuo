Context View: Async State Management

Tenuo solves the "Invisible State" problem in async Python frameworks. ContextVar propagates through await boundaries, making warrant context available to all @lockdown functions automatically.

1. Request Arrives
HTTP request arrives at FastAPI/LangChain application. Warrant may be in header or loaded from pod config.
ContextVar: None
2. Middleware Sets Context
FastAPI middleware extracts warrant and sets it in ContextVar. This propagates through all async operations.
with set_warrant_context(warrant): # ContextVar now set

3. LangChain AgentExecutor
AgentExecutor processes request. ContextVar automatically propagates through await boundaries. No manual passing needed.

üîë Key Insight: Invisible State Made Visible

In async Python frameworks, function calls happen across await boundaries, making it hard to pass context. Tenuo uses Python's ContextVar which automatically propagates through async operations. Set the warrant once in middleware, and all @lockdown functions have access to it - no manual passing required.
