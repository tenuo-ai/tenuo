Tenuo + LangChain
Zero-Import Authorization for AI Agents
Protect your LangChain tools and LangGraph workflows with cryptographically-enforced authorization. Your tool code stays clean ‚Äî Tenuo wraps at setup time.
üéØ Core Principle: Zero Tenuo Imports in User Code

Tools and nodes are pure business logic. Authorization is configuration. Wrap once at setup, protect everywhere.

Two Integration Patterns

Choose based on your architecture

üîß
Single Agent: protect_tools()
Wrap tools at setup time
Recommended

For standard LangChain agents. Your tools are plain functions ‚Äî protect_tools() wraps them with authorization. 

# Tools: Plain Python functions - NO Tenuo imports!
def read_file(file_path: str) -> str:
    return open(file_path).read()

def search(query: str) -> str:
    return f"Results for: {query}"

# Setup: Wrap once with authorization
from tenuo.langchain import protect_tools

secure_tools = protect_tools(
    tools=[read_file, search],
    warrant=root_warrant,
    keypair=keypair,
    config={
        "read_file": {"constraints": {"file_path": {"pattern": "/tmp/*"}}}
    }
)

# Use with LangChain
agent = AgentExecutor(agent=base_agent, tools=secure_tools)
  • ‚úì Tools stay pure business logic
  • ‚úì Config-based per-tool constraints
  • ‚úì Automatic PoP signature handling
  • ‚úì Full audit logging
üîÄ
Multi-Agent: SecureGraph
Automatic warrant flow in LangGraph
Advanced

For LangGraph multi-agent workflows. SecureGraph automatically attenuates warrants per-node with dynamic constraints. 

# Graph: Standard LangGraph - NO Tenuo in nodes!
graph = StateGraph(AgentState)
graph.add_node("supervisor", supervisor_fn)
graph.add_node("researcher", researcher_fn)
graph.add_node("writer", writer_fn)

# Setup: Wrap with automatic warrant management
from tenuo.langgraph import SecureGraph

secure = SecureGraph(
    graph=graph,
    config={
        "nodes": {
            "researcher": {
                "attenuate": {
                    "tools": ["search", "read_file"],
                    "constraints": {
                        "path": {"pattern": "/data/${state.project_id}/*"}
                    }
                }
            }
        }
    },
    root_warrant=warrant,
    keypair=keypair
)

result = secure.invoke({"input": "...", "project_id": "alpha"})
  • ‚úì Dynamic ${state.*} interpolation
  • ‚úì Automatic warrant attenuation per-node
  • ‚úì Stack-based delegation (push/pop)
  • ‚úì Input validation prevents injection

Dynamic Constraints (SecureGraph)

State values are interpolated into constraints at runtime with security validation. 

# Config with dynamic constraint
config = {
    "nodes": {
        "file_reader": {
            "attenuate": {
                "constraints": {
                    "path": {
                        "pattern": "/uploads/${state.user_id}/*",
                        "validate": "^[a-zA-Z0-9_-]+$"  # SECURITY: Prevents injection
                    }
                }
            }
        }
    }
}

# At runtime with state = {"user_id": "alice"}
# Constraint becomes: Pattern("/uploads/alice/*")

# If attacker tries state = {"user_id": "../../../etc"}
# Validation fails ‚Üí InterpolationValidationError
‚ö†Ô∏è Security: Validation is Required

Always include a validate regex when using ${state.*} interpolation. Without validation, path traversal attacks like ../../../etc/passwd are possible. Missing state keys also fail-safe with InterpolationValidationError.

Security Features

Warrants bound to holder's public key. Stolen warrants are useless without the private key.

100% local verification in ~25Œºs. No network calls, no single point of failure.

Quick Start

# 1. Install
pip install tenuo

# 2. Define tools (plain functions)
def read_file(file_path: str) -> str:
    return open(file_path).read()

# 3. Create warrant
from tenuo import Keypair, Warrant, Pattern

keypair = Keypair.generate()
warrant = Warrant.create(
    tool="read_file",
    constraints={"file_path": Pattern("/tmp/*")},
    ttl_seconds=3600,
    keypair=keypair,
    authorized_holder=keypair.public_key()  # PoP binding!
)

# 4. Protect tools
from tenuo.langchain import protect_tools

secure_tools = protect_tools([read_file], warrant=warrant, keypair=keypair)

# 5. Use with LangChain
from langchain.agents import AgentExecutor
agent_executor = AgentExecutor(agent=agent, tools=secure_tools)
response = agent_executor.invoke({"input": "Read /tmp/test.txt"})
