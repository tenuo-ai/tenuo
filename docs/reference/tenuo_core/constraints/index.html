Constraint types for Tenuo warrants.

Constraints restrict what argument values are allowed when a tool is invoked. The key invariant is monotonicity: child constraints can only be stricter than parent constraints, never looser.

Â§Security: Constraint Depth Limit

Recursive constraint types (All, Any, Not) are limited to a maximum nesting depth of MAX_CONSTRAINT_DEPTH (16) to prevent stack overflow attacks from maliciously crafted warrants.
