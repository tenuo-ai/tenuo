Use Cases

üîß Tool Authorization
Control which tools an agent can invoke and with what parameters. "Agent X can call query_db but only on public_* tables."
üí∞ Budget Limits
Enforce spending caps across agent hierarchies. Parent grants $10k, orchestrator limits to $1k per worker.
üîê Multi-Sig Approvals v0.2
Sensitive actions require M-of-N human approval. "Delete requires 2-of-3 admins. Approvals bound to holder."
ü§ñ Multi-Agent Orchestration

üèóÔ∏è Infrastructure Access
Kubernetes cluster operations, cloud resource management. "Upgrade staging-web only, not production."
‚è±Ô∏è Time-Bounded Tasks
